项目介绍
本项目是针对帕金森诊断多活动组合进行建立的。
能够将不同活动的特征进行合并，构建组合后的特征文件。包含生成.csv文件


模块功能介绍
1、choose_activity(sequence,samples)
该模块的功能是拼接指定活动，输入需要的活动序号以及保存的样本数量，
输出拼接好的特征文件。
'''
       Describe
       选择指定活动进行拼接
       -----------------
       Parameters
       -----------------
       data:原始特征文件
       sequence：活动号序列
       samples:样本数量
       severity_level：患病等级列数字
       subject_id:病人编号所在列的数字
       info1:保存疾病等级信息
       info2:保存病人序号信息
       
       Return
       -----------------
       df6：活动拼接后的csv文件      '''


执行流程：
1、选择活动：请输入所选择的活动序号

       sequence=[2,3]#如果需要选择活动2和3

2、选择样本条数（根据滑动窗口大小调整样本数量）

        samples = 24 #滑动窗口大小为300，样本数为24；如窗口大小为600，样本数为12

3、进行活动横向拼接

     choose_activity(sequence,samples)

     正在进行活动拼接...
     已成功拼接活动，生成活动2+3拼接后特征文件

4、输出拼接后特征文件

     output_Feacom_file()
